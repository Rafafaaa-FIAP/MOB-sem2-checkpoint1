# 🏆 Interclasse Digital

## 📖 Sobre o Projeto
O **Interclasse Digital** é uma aplicação mobile desenvolvida para a gestão e acompanhamento de um torneio interturmas. O objetivo principal é fornecer uma plataforma interativa onde os utilizadores podem visualizar as modalidades desportivas, a tabela de classificação das turmas e o calendário dos jogos.

Este projeto é referente ao 2º Semestre (CP) da disciplina de Desenvolvimento Mobile da FIAP, lecionada pelo Prof. Adeilton Meneses.

## 🚀 Funcionalidades Principais
A aplicação contempla as seguintes funcionalidades essenciais exigidas para o projeto:
* **Modalidades:** Consulta dos desportos disponíveis no torneio, como Futsal, Vôlei, Basquete e E-sports.
* **Tabela de Classificação:** Visualização da pontuação em tempo real das salas participantes (ex: 3º ES Paulista, 2º ES Lins, 1º ADS).
* **Calendário de Jogos:** Acesso aos horários e locais das partidas, com a informação a ser guardada de forma persistente através do `AsyncStorage`.

## 🛠 Tecnologias e Estrutura da Aplicação
A aplicação foi construída utilizando os seguintes recursos tecnológicos e de interface:
* **Componentes Nativos:** Utilização de componentes base como Texto, Input e Botão.
* **Estilização:** Aplicação de estilos através de `StyleSheet`.
* **Integração com API:** Implementação realizada com a biblioteca `Axios`, englobando a criação e o tratamento de:
    * 1 API GET
    * 2 APIs POST (com os respetivos parâmetros e tratamentos)
